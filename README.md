
# 📉 Frizz: The Anti-Rizz AI

**Frizz** is a social defense tool designed to transform high-energy, "rizz-heavy," or unwanted messages into dry, professional, and conversation-ending corporate-speak.

## 🚀 The Vision
In an era of AI-generated "rizz," Frizz provides the ultimate counter-measure. Whether you're dealing with a cringey DM or just want to end a conversation without being rude, Frizz analyzes the context and provides the most boring response possible.

## ✨ Features
- **Text-to-Bore:** Instant transformation of flirty/fun text into bureaucratic formal-speak.
- **Contextual Screenshot Analysis:** Upload a chat screenshot; Frizz reads the vibe and generates a vibeless reply.
- **Boredom Slider:** Choose your level of dullness, from "Office Manager" to "Legal Department."

## 🛠️ Technical Stack
- **Language:** Python
- **Frontend:** Streamlit
- **AI Brain:** Hack Club AI API (GPT-4o / Gemini 1.5 Pro)
- **Tracking:** Logged via Hackatime ⏱️

## 🔧 Installation & Setup
1. Clone the repo: `git clone https://github.com/RiyaDhami13/frizz-ai.git`
2. Install dependencies: `pip install streamlit requests`
3. Run the app: `streamlit run app.py`
